Microsoft SQL Server 2008 自述文件

2008 年 6 月 27 日下午 4:43:51

此自述文档包含了重要信息，请在安装 Microsoft® SQL Server® 2008 之前先阅读。本文档介绍了如何在《联机丛书》中查找安装帮助，如何查找“升级顾问”，如何在“发行说明”中查找补充的产品信息，另外还介绍了产品的版权归属。

本自述文件在安装媒体上位于 <驱动器>:\，也可以通过自述文件下载页获得。

使用联机丛书获取安装帮助

SQL Server 2008 联机丛书是获取关于 SQL Server 安装帮助的主要文档。可以通过以下方式访问联机丛书：

  • 使用为这一版 SQL Server 2008 提供的文档安装选项安装《联机丛书》。安装之后，更新到最新版。最新版可从《联机丛书》下载页面得到。
  • 在 MSDN 上查看联机丛书。
  • 通过联机丛书下载页下载独立版本。

若要在不访问联机丛书的情况下查看联机丛书安装文档，可以从安装媒体或从计算机使用以下路径打开 s10ch_setup.chm 文件：

  • 若要从安装媒体打开 s10ch_setup.chm，请打开 <驱动器>:\[x86] | [x64] | [ia64]\help\2052\s10ch_setup.chm。
  • 若要从计算机打开 s10ch_setup.chm，请打开 <驱动器>:\%Program Files%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\2052\s10ch_setup.chm。

通过升级顾问对升级进行分析

升级顾问工具分析当前的 SQL Server 实例，提供了应在升级前掌握的重要信息。有关 SQL Server 2008 升级顾问的详细信息，请参阅升级顾问网站。

发行说明提供了有关仅适用于此版本或未在此版本的联机丛书中记录的产品行为的其他补充信息。可通过发行说明下载页下载发行说明文档。
